# Итоговое задание по программной инженерии

Чтобы запустить оба проекта с помощью Docker, нужно ввести команду:

```
sudo docker-compose up --build
```

И потом открыть в браузере `http://localhost:8000/docs`.

Также можно запустить сервис локально, без Docker.

Для этого нужно запустить команду:

```
sudo uvicorn main_local:app --reload --host 0.0.0.0 --port 80
```
